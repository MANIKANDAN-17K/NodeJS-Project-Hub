## ⚙️ Installation & Setup

### 1️⃣ Clone repository

```
git clone <repo-url>
cd <project-folder>
```

### 2️⃣ Install dependencies

```
npm install
```

### 3️⃣ Setup environment variables

Create `.env` file:

```
PORT=5000
DB_URI=your_database_url
SECRET_KEY=your_secret
```

### 4️⃣ Run server

```
npm start
```

For development mode:

```
npm run dev
```
